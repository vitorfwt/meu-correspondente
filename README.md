# Meu Correspondente

Projeto "Meu Correspondente" estruturado como um monorepo.

## Estrutura do Projeto

O projeto está dividido nas seguintes pastas:

- **`api/`**: Backend em Node.js (TypeScript + Vitest).
- **`crawler/`**: Scripts para captura e extração de dados.
- **`app/`**: Aplicativo mobile em Flutter (Frontend).

## Requisitos

- Node.js (v18+)
- Flutter SDK (para rodar o aplicativo móvel)

## Como rodar o Backend

### Instalação de Dependências

Navegue até a pasta `api/` e instale as dependências:

```bash
cd api
npm install
```

### Rodar Testes

Para rodar a suíte de testes unitários do backend:

```bash
npm run test
```
